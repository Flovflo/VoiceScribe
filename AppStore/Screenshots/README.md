# App Store Screenshot Plan

Create real screenshots from the App Store build after launching the app from Xcode.

## Required captures

1. HUD idle state
2. HUD recording state
3. Settings showing:
   - selected Qwen3-ASR model
   - transcription language picker
4. Menu bar interaction with the status menu open

## macOS App Store notes

- Use real app UI, not marketing mockups.
- Avoid placeholder transcript text or fake account data.
- Capture on a clean desktop with readable text.
- Keep the screenshots focused on the Mac experience.

## Suggested transcript text

Use short neutral sample text such as:

- "Draft meeting summary for Tuesday afternoon."
- "Remind me to send the project update before six."

## Submission checklist

- Verify the App Store build is the one running.
- Verify the clipboard-only behavior in the App Store build.
- Ensure the microphone permission prompt text matches the shipped copy.
- Save final assets in this folder before uploading them to App Store Connect.
