## Common help topics

### Microphone access

If transcription does not start, verify microphone access in macOS System Settings.

### Model download

The first launch can download a local MLX speech model. This requires network access and enough disk space for the selected model.

### Clipboard behavior

- Standard distribution can auto-paste after transcription.
- App Store distribution copies the transcript to the clipboard for manual paste.

### Permissions

If the app cannot interact as expected, remove and re-grant the relevant macOS permissions, then relaunch the app.
